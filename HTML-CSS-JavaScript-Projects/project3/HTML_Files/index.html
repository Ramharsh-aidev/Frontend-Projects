Binary Search Algorithm

Discover the Binary Search algorithm, an efficient method for finding an item in a sorted array. Learn how it works through a divide-and-conquer approach, significantly reducing the time complexity compared to linear search.

LinkedList

Stack

Understand the fundamentals of Stacks, it follows Last In, First Out (LIFO) principle. Elements are added and removed from the top, making it ideal for tasks like undo operations and function calls.
